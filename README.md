### Tutor Algébrico

<img align="right" src="https://raw.githubusercontent.com/khrir/khrir/main/assets/tutor_demo.gif" width="140" alt="Tutor Algébrico Demo"/>

*Offline-first PWA that guides high school students through algebra step-by-step,
with metacognition tracking and teacher reporting.*

**Stack:** TypeScript · React · Vite · Tailwind CSS · Python · FastAPI · Alembic · Docker

- Step-by-step algebra solver that validates each transformation (distributive,
  combining terms, isolating variables) and surfaces targeted hints on failure
- Confidence tracking before and after each problem — supporting metacognitive habits
- Fully offline: IndexedDB session persistence, Service Worker caching, and
  automatic sync queue when connectivity returns
- Teacher dashboard with error-category breakdown, average time per step, and CSV export

<br clear="right"/>

### Hawke

*Full-stack event management platform — scheduling, ticketing, payments, and invoicing.*

**Stack:** Python · REST API · Clean Architecture · Docker

- Complete domain model: Events, Activities, Speakers, Lots, Purchases, Payments, Invoices
- Tiered ticket sales (lots) with availability control and multi-payment-method support
- Role-based access for organizers and participants
- Clean architecture with separated entities, repositories, and builders — built from
  hands-on experience working at an event platform (Doity)
